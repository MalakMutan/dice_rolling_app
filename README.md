# 🎲 Dice Rolling App

A simple Flutter app that simulates rolling a dice with a random outcome every time you tap.  
This project is perfect for beginners to learn about Flutter widgets, state management, and working with images.

---

## 🚀 Getting Started

This project is a beginner-friendly Flutter application that demonstrates:

- `setState()` usage for UI updates
- Random number generation
- Displaying assets and images

---

## ✅ Prerequisites

Before running this project, ensure you have the following installed:

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- Dart SDK (included with Flutter)
- IDE (VS Code / Android Studio)
- A connected emulator or physical device

---

## 🛠️ How to Run the App

1. **Clone the repository**

```bash
git clone https://github.com/your-username/dice_rolling_app.git
cd dice_rolling_app
````

2. **Get the dependencies**

```bash
flutter pub get
```

3. **Run the app**

```bash
flutter run
```
